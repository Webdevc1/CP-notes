# Miller-Rabin Primality Test ([[MRPT]1])

It is a direct extension of the Sieve of Eratosthenes, with two more equations added to the fold.
Just like SOE1, MRPT1 doesn't actually guarantee that a number is prime, well not until you check for all the numbers,
but that defeats the purpose of using the test.

It uses Euclid's lemma, with SOE1. 

                                            The time complexity of the test is O(log3 n).
