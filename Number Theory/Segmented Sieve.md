# Segmented Sieve (SS1) ([[SOE1]1])

SS1 is a method completely derived from SOE1, like the name suggests it just a layer of segmentation applied to the SOE1 or the vector.

In this method, we simply break the original vector of size n (the number till which we find the prime numbers or the highest number given) into segments.
It's a convention to take the segment length as sqrt(n). 

  The Time complexity of SS1 is  $O(n ln ln  \(\sqrt{n}\) ) \approx  O(n ln ln n) $.
